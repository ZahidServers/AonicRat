**Version:** 1.0.0-beta

---

## What is AonicRAT?

AonicRAT is a Java-based Remote Administration Tool (RAT) built for **educational and demonstration purposes only**.  
It is intended to help students, researchers, and security professionals understand how remote access tools work at a technical level.

---

## Features

- File System browser (remote file browsing, upload, download)
- Command Line / Terminal (persistent shell session)
- GUI Remote (remote desktop viewing)
- Network Controls
- Client Generator (generate agents for Windows, Linux, macOS)
- Advanced controls
- Settings

---

## Requirements

- Java 18 or higher (on the server/controller machine)
- Java 11 or higher (on the target/agent machine)

---

## Building

This is a Maven project. Open in Apache NetBeans and run:

```
Run → Clean and Build Project
```

The executable fat JAR will be generated at:

```
target/AonicRat-1.0.0-beta.jar
```

Run with:

```bash
java -jar target/AonicRat-1.0.0-beta.jar
```
