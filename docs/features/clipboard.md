# Clipboard

![Clipboard](../Screenshot_10.png)

Remote clipboard access — read and write the clipboard of the target device.

---

## Capabilities

- **Get Clipboard** — retrieves the current clipboard content from the remote machine
- **Set Clipboard** — pushes custom text to the remote machine's clipboard

---

## What You Learn

!!! info "Industry Comparison"
    This is how **TeamViewer's clipboard sync**, **AnyDesk clipboard sharing**, and remote support tools implement bidirectional clipboard transfer. Understanding clipboard access is also relevant to endpoint security — clipboard monitoring and protection is a feature of many DLP (Data Loss Prevention) and EDR solutions.
