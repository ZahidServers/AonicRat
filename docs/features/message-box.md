# Message Box

![Message Box](../Screenshot_14.png)

Send a pop-up message box / alert dialog to the remote device's screen.

---

## Capabilities

- **Title** — set the dialog title (e.g., "Alert", "System Notice")
- **Message** — compose the message body
- **Type** — choose the dialog type:
    - Information
    - Warning
    - Error
    - Question
- **Send Message Box to Remote Device** — delivers the dialog instantly

---

## What You Learn

!!! info "Industry Comparison"
    Enterprise IT helpdesk tools and system administrators use remote message boxes to **broadcast on-screen notifications** to managed machines — scheduled maintenance alerts, security warnings, and IT communications. This is also how the Windows `msg` command works, and how enterprise tools like **PDQ Deploy** and **ManageEngine** send operator notifications to end-user machines.
