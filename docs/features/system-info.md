# System Info

![System Info](../Screenshot_8.png)

Retrieves and displays comprehensive system information from the remote device.

---

## Capabilities

Displays key system properties in a Name/Value table:

- OS name, version, and architecture
- CPU model and core count
- Total and available RAM
- Current username and hostname
- Java runtime version on the remote machine
- System uptime
- Other environment properties

---

## What You Learn

!!! info "Industry Comparison"
    This is how enterprise **asset management tools** (ManageEngine Asset Explorer, Microsoft SCCM, Lansweeper) and penetration testing frameworks perform **system fingerprinting** and **enumeration** on remote hosts — gathering OS version, hardware specs, and user context before deciding what actions to take.
